# Merge Tactics Deck Optimizer

A small web tool to help build 6-card boards that have 6 traits (maximum) active. 

<img src="./images/website-image.png" alt="Demo" width="500" height="333">

You can:
- Click cards on a grid to select them
- See which other cards are still viable, given your current board
- See how many possible 6-trait decks remain that include your current picks
- Reset your selection with one click in the top right corner
